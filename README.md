
# 这是利用openwechat开发的一个聊天机器人
# 能做什么
## 记录灵感和知识碎片
将自己与机器人放在群里后
可以发送文字，语音，文件，图片，都会被记录到data文件夹下创立的日记markdown文件下
发送微信公众号链接，链接标题，地址和摘要会被记录下来
发送视频号，会把视频号标题和摘要记录下来
输入暗语，暗语可以在config.json里配置，会自动在日记文件中创立分割线
## 调用deepseek
config.json里配置deepseek的API key
@机器人，可以问问题，会调用deepseek回答
回答都会记录到 日志文件下



# 需要什么
## 安装
可以打包成docker脱管
数据在data里
配置在config.json里
## 额外需求
微信小号1个
和你自己微信号加成群，群民在config.json里配置好
微信不支持两人加群，可以请第三人帮忙，然后再把第三人退出
